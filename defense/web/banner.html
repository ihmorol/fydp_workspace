Introduction & Motivation

Coupled nonlinear ODEs like the Lorenz-1960 system have no closed-form solution. We compare two distinct solvers:

Runge-Kutta (RK4)
  • Step-by-step solver
  • Mesh-dependent grid
  • Redoes integration for every query
Plain ANN
  • Direct mapping: t → (x,y,z)
  • Continuous & mesh-free
  • Instant one-shot query
Central Question: Can a plain neural network, trained only on data with no physics in the loss, learn this coupled system, and which shape does it best?
02

Objectives

1. Ground Truth
Build a trusted reference using custom RK4 and SciPy solvers, cross-checking their agreement.
2. Search Space
Vary depth, width, and activation functions systematically with zero physics in the loss.
3. Performance
Evaluate models on error (MSE, MAE, Max Error) and physical cost (train & inference time).
4. Literature Check
Benchmark the best plain ANN configurations against published physics-informed results.
03

The System

General Form
dx/dt = kl(1/(k²+l²) - 1/k²) yz
dy/dt = kl(1/l² - 1/(k²+l²)) xz
dz/dt = (kl²/2)(1/k² - 1/l²) xy
Reduced Form
ẋ = -0.10 yz
ẏ = +1.60 xz
ż = -0.75 xy
Parameters
k = 2, l = 1
Initial Values
x₀=0.5, y₀=0.75, z₀=1
Time Span
t ∈ [0, 1]

These coupled nonlinear ordinary differential equations represent meteorological fluid circulation. They are compact to write, but highly sensitive to numerical integration precision.

Quasi-periodic trajectories over t ∈ [0, 50] (long-term).
